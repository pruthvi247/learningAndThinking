## odered and unodered list

### Unodered list

```html
<ul>
<li>element1</li>
<li>element2</li>
</ul>
```
**output
<ul>
<li>element1</li>
<li>element2</li>
</ul>
### odered list

```html
<ol>
<li>element1</li>
<li>element2</li>
</ol>
```
**output
<ol>
<li>element1</li>
<li>element2</li>
</ol>
